# Svelte, Tiptap Notion Clone

### Running Locally

```bash
git clone https://github.com/andheller/svelte-tiptap
cd svelte-tiptap
pnpm i
pnpm dev
```
### Running Locally with Docker 🐳

```bash
docker compose up
```

App will be available at `localhost:3002`
